In createOuterShellGeometries, delete the if (svgIsClickerShape) { ... } else { ... } branching entirely. Replace with a single code path that always treats the SVG as the clicker body:
tsconst CLEARANCE = settings.clearanceMm ?? DEFAULT_SETTINGS.clearanceMm;

// SVG = clicker body. Pocket = SVG + clearance. Outer wall = pocket + wall thickness.
const innerShape = expandShapeOutward(cloneShape(svgShape), CLEARANCE);
